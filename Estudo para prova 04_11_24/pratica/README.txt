Olá! Aqui é Pedro. Em preparação a uma prova de POO (Programação Orientada a Objetos) eu resolvi fazer um estudo, utilizando as tecnicas já ensinadas em sala.
Criei, a partir dos meus conhecimentos, uma estrutura de universidade, onde podemos inserir e checar dados pertinentes para alunos, disciplinas, cursos, professores e etc.

#Alunos
Em alunos eu determinei seu número de matricula e nome. Todas as informações sensiveis estão no escopo privado (encapsuladas).

#Professores
Determinei um enum para professores. Este enum indica qual o nivel de escolaridade do professor.

#Curso
Os cursos tem coordenadores.

#I_dados e dados
Criei uma interface para posteriormente o pseudo banco de dados utilizar (implementar) as funções que determinei.

#Impressão e Executar

Separei as funções de imprimir e executar para deixar o código mais fácil de ler. Em informações do curso podemos acessar os alunos, as disciplinas, o coordenador de curso e professores.
A impressão usa a classe Informacao_curso como base para a impressão de texto no console.
